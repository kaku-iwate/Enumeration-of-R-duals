大学毕业的研究课题 -- 矩形对偶的列举算法.(其实就是把别人论文提到的算法实现一遍 = =)

语言为common lisp,使用lispbox编译,直接把lisp文件拖进Emacs窗口,按下 Esc 再按下 X 输入 Slime 来打开slime窗口,随后按下 Ctrl+C Ctrl+K 和 Ctrl+C Ctrl+L 即可.

输入为三角分割图形的各个边的数据(包括边的两个顶点和方向)以及论文中所提到的T构造.r-dv1.lisp文件里用 ; 来注释的部分是两个输入示例.

学习common lisp是读了《实用Common Lisp编程》,读了一半能支持写完这个算法就没继续了.这门语言没啥人用,了解一下就好了,继续深入学习没法产生价值 = =.
